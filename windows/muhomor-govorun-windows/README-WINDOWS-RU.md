# Мухомор - Говорун для Windows

Упрощенная Windows-версия с теми же тремя режимами озвучки:

- `Локальная простая быстрая модель` — встроенный Windows SAPI, без API-ключей.
- `SaluteSpeech быстрый облачный` — нужен `salute.auth_key` в `config.json`.
- `ElevenLabs красивый голос` — нужен `elevenlabs.api_key` в `config.json`.

## Запуск

1. Установите Python 3 для Windows, если он еще не установлен.
2. Откройте папку приложения.
3. Запустите `start.bat`.
4. При первом запуске рядом появится `config.json`.
5. Впишите API-ключи в `config.json`, если нужны SaluteSpeech или ElevenLabs.

## Горячие клавиши

- `Ctrl+Shift+V` — озвучить текст из буфера обмена.
- `Ctrl+Shift+S` — остановить озвучку.

## Важно

Ключи не входят в архив. Храните их только в локальном `config.json`.
