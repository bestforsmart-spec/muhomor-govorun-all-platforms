## Description

ShromSpeak helps you turn typed text into spoken audio on iPhone.

Use it when you want a simple voice for quick phrases, copied text, or everyday communication. Type or paste text, tap Speak, and your iPhone reads it aloud using local system voices.

Core features:

- Offline text-to-speech using local iOS voices
- Fast text input for phrases and messages
- Speak, stop, and paste-from-clipboard controls
- Automatic language detection
- Adjustable speech speed and pitch
- Light and dark themes
- Share/action extension for selected text

ShromSpeak is designed as a simple assistive communication tool. It can help people who temporarily or permanently prefer typed speech in everyday situations.

The current version does not use cloud voice services and does not require an internet connection for speech playback.

## Review Notes

ShromSpeak is an offline local text-to-speech app. It uses AVSpeechSynthesizer and local iOS system voices. No account is required. No cloud service is used in version 1.0.

To test:

1. Open the app.
2. Type or paste text.
3. Tap Speak.
4. Use Stop to interrupt playback.
5. Open Settings to adjust speech speed, pitch, and theme.

The Share/Action extension is named "Озвучить" and appears for selected/shared text.
