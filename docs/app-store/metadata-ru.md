# Метаданные App Store для ShromSpeak

## Название

ShromSpeak

## Подзаголовок

Офлайн-голос для текста

## Promotional Text

Введите или вставьте текст, и iPhone озвучит его локальным голосом без интернета.

## Описание

ShromSpeak превращает набранный текст в голос на iPhone.

Приложение удобно для быстрых фраз, скопированного текста и повседневного общения. Введите или вставьте текст, нажмите "Озвучить", и iPhone произнесет его локальным системным голосом.

Возможности:

- офлайн-озвучка текста локальными голосами iOS;
- быстрый ввод фраз и сообщений;
- кнопки "Буфер", "Озвучить" и "Стоп";
- автоматическое определение языка;
- настройка скорости и тона речи;
- светлая и темная тема;
- расширение "Озвучить" для выделенного или переданного текста.

ShromSpeak задуман как простой инструмент альтернативной коммуникации. Он может помочь людям, которым удобнее говорить через набранный текст в повседневных ситуациях.

В версии 1.0 не используются облачные голосовые сервисы, а озвучка не требует подключения к интернету.

## Заметки для ревью

ShromSpeak - офлайн-приложение для локальной озвучки текста. Используется AVSpeechSynthesizer и системные голоса iOS. Аккаунт не нужен. Облачные сервисы в версии 1.0 не используются.

Проверка:

1. Откройте приложение.
2. Введите или вставьте текст.
3. Нажмите "Озвучить".
4. Нажмите "Стоп", чтобы остановить речь.
5. Откройте настройки, чтобы изменить скорость, тон и тему.

Расширение Share/Action называется "Озвучить" и появляется для выделенного/переданного текста.
