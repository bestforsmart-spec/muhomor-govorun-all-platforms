# Мухомор - Говорун для iPhone

Локальная офлайн-версия для iPhone.

## Что реализовано

- Основное приложение для ручной вставки текста и озвучки.
- Share Extension с названием `Озвучить`.
- Озвучка работает локально через `AVSpeechSynthesizer`.
- Интернет, SaluteSpeech, ElevenLabs и API-ключи не используются.

## Реальный сценарий на iOS

iOS не разрешает стороннему приложению добавить свою кнопку прямо в системное меню выделения рядом с `Copy/Paste` во всех приложениях.

Доступный системный сценарий:

1. Выделить текст в приложении, где есть системное меню выделения.
2. Нажать `Поделиться` / `Share`.
3. Выбрать действие `Озвучить`.
4. Расширение откроется и начнет говорить локальным голосом iPhone.

Если `Озвучить` не видно сразу, его нужно включить в списке действий Share Sheet через `Edit Actions` / `Изменить действия`.

## Офлайн-условие

Для полностью офлайн-работы на iPhone должен быть установлен русский системный голос:

`Settings` -> `Accessibility` -> `Spoken Content` -> `Voices` -> `Russian`

Если голос не установлен, iPhone может потребовать скачать его один раз.

## Установка на телефон

1. Установить Xcode.
2. Открыть `ios/MuhomorGovorunIOS.xcodeproj`.
3. В Xcode выбрать target `MuhomorGovorunIOS`.
4. В `Signing & Capabilities` включить `Automatically manage signing`.
5. Выбрать свой Apple ID/team.
6. Подключить iPhone.
7. Выбрать iPhone как Run Destination.
8. Нажать `Run`.

После установки расширение `Озвучить` появится в Share Sheet.
