# Мухомор - Говорун для iPhone

SwiftUI MVP для iPhone с тремя режимами озвучки:

- локальный быстрый `AVSpeechSynthesizer`;
- SaluteSpeech быстрый облачный;
- ElevenLabs красивый голос.

## Сценарии

- вставить текст из буфера;
- выбрать голос;
- озвучить;
- остановить озвучку;
- сохранить API-ключи локально в `UserDefaults`.

## Ограничения iOS

iOS не разрешает обычному приложению глобально читать выделенный текст из других приложений или вешать системные хоткеи как на macOS/Windows. Для полноценного сценария “выделил в другом приложении -> озвучил” следующей итерацией нужен Share Extension.

## Сборка

Откройте проект `MuhomorGovorunIOS.xcodeproj` в Xcode и запустите на iPhone или Simulator.

В этой Codex-сессии IPA не собран: установлен только Command Line Tools, полноценного Xcode toolchain для `xcodebuild` нет.
