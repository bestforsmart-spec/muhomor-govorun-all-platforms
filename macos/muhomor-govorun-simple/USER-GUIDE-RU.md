# Пользовательская инструкция

`Мухомор - Говорун simple` живет в верхнем меню macOS.

## Как пользоваться

1. Выделите текст или скопируйте его.
2. Откройте меню `Мухомор - Говорун`.
3. Выберите `Озвучка текста` или `Саммари и озвучка`.
4. При необходимости нажмите `Стоп озвучка`.

## Выбор голоса

Откройте `Голос` и выберите один из трех режимов:

- локальный быстрый голос;
- быстрый облачный SaluteSpeech;
- красивый голос ElevenLabs.

Для SaluteSpeech и ElevenLabs нужны API-ключи. Их можно добавить в `Настройки`.
